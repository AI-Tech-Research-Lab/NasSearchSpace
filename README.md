# NasSearchSpace

Repo containing different search spaces for NAS:
- Once-For-All (OFA) MobilenetV3
- OFA Resnet50
- NASBench201
- DARTS

[3] DARTS: Differentiable Architecture Search (https://arxiv.org/abs/1806.09055)
